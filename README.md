# ⚡ FastBiller

A fast and intuitive billing application designed to **eliminate traditional form-based workflows** and let users create bills directly on the final bill layout.

---

## 🚀 The Idea

Most billing systems follow a slow process:
> Fill forms → Preview → Print

This breaks user flow and wastes time.

QuickBiller solves this by allowing users to:
> **Create and edit bills directly on the final bill view (WYSIWYG)**

What you see is exactly what gets printed.

## ⚙️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Node.js  
- **Database:** MongoDB  

---

## 🔍 Key Features

- 🧾 Direct bill editing on final layout (print-ready)
- 📦 Product management (Items window)
- 👥 Customer management (Parties window)
- 📊 Dashboard analytics (Line, Bar, Pie charts)
- 📈 Sales & receipts tracking (custom timeline)
- 📒 Party ledger system
- ⚙️ Business date configuration

---

## 🧠 Approach

Instead of separating input and output, QuickBiller uses:
- Direct interaction with the final document
- Real-time updates
- Minimal steps for faster billing

---

## ⚠️ Challenges

## ▶️ How to Run

### Prerequisites
- MongoDB must be installed and running locally

---

### Setup Steps

1. Clone the repository  
```bash
git clone https://github.com/ShreySanja/Fast-Biller.git
```

2. Install dependencies
For backend:
```bash
cd fastbiller-server
npm install
```
For frontend:
```bash
cd fastbiller
npm install
```

3. Start the application

Run backend:
```bash
node server.js
```
Run frontend:
```bash
npm start
```

4. Open in browser
```bash
http://localhost:3000
```
